STATS KERNEL DENSITY Extension Command

Produce a dataset of the smoothed density, a summary table, and an optional plot.

STATS KERNEL DENSITY VARIABLES=variables*
KERNEL=EPANECHNIKOV** or GAUSSIAN or RECTANGULAR or TRIANGULAR or BIWEIGHT or COSINE or OPTCOSINE
OUTPUTDS=dsnames N=integer BANDWIDTHFACTOR=number

/OPTIONS PLOT=YES** or NO

* Required
** Default

STATS IRM /HELP prints this information and does nothing else.

STATS KERNEL DENSITY VARIABLES=y1 y2
OUTPUTDS=y1dataset y2dataset.

VARIABLES specifies the variables whose distributions will be calculated. Missing values are automatically discarded separately for each variable.

KERNEL specifies the kernel to be used for smoothing.

BANDWIDTHFACTOR specifies a multiplier for the automatic bandwidth. The default is 1.

N specifies at how many points the probability is calculated. The default is 512.

OUTPUTDS specifies the names for new output datasets to hold the calculated values. If specified, there must be as many dataset names as variables. The dataset names must not already be in use. There will be N cases in each dataset. If OUTPUTDS is not specified, no datasets will be created.

If any new datasets are created, and the active dataset is unnamed, it will be closed automatically as per standard rules.

OPTIONS

PLOT specifies whether or not to plot the density function. A separate plot is produced for each variable.
